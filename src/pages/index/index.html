What is Presto Parts?

A Dramatization:

Your concert band is playing songs by "The Insects" for the spring concert.

Your arranger sent you PDFs of each piece.

To facilitate printing and distributing music, you want to rearrange the PDFs so that each file has each player's individual part for the whole concert.

You open Presto Parts.

First, you to separate each song PDF into its individual parts.

For each piece, you import the song PDF and list out each part and its number of pages.

The result is a folder for each piece containing PDFs for all of its individual parts:

Now, you glue each like-part together.

You list each song in concert order.

You list each part and its number of players.

The final result is a PDF for each musician, plus a "MASTER" PDF containing your specified number of copies of each part, ready to print, all in one!
